Relationship between nominal and ordinal features

카이제곱 검정
명목척도(nominal)를 이용하여 측정된 자료에서 연관성을 찾을 때 사용.
상관성은 서열척도(ordinal), 구간척도, 비율척도로 측정된 자료의 연관성을 찾을 때 사용.
