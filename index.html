AngularJS 정리

Attribute Directive 정리

ng-model
: input tag 에서 사용

ng-bind
: DOM tag 내의 text를 {{ }} 로 적용해서 하는 것과 동일.
angular가 다 로드 되기 전에 {{ }} 구문이 표시될 수 있음.
이 때 사용함. ng-cloak도 확인 필요.

Module

서로 관련 있는 코드를 묶는 단위. Java에서 패키지와 같은 것이라 이해하면 됨.
모듈은 controller, service, factory, directive 등을 포함할 수 있고, 서로 간의 연관 관계를 정의할 수 있다.
애플리케이션 역시 모듈이며 ng-app에 애플리케이션 모듈의 이름을 설정하여 사용한다.

Controller

$scope 대신에 Controller 정의 함수의 this를 통해 변수를 정의할 수 있음.

이름을 축약해서 사용할 수 있음.
