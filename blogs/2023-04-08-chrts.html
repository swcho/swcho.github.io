CHRTS 개념

Data visualization의 CHRTS가 무엇인지 그 예시들도 찾아 봅니다.

2023-04-08

CHRTS

Amazon.com: 데이터 시각화: 데이터 기반 디자인 핸드북(영어판).: 9781473912144: Kirk, Andy: 도서

  • Categorical: 카테고리 값들의 비교, 정량값의 분포를 비교
  • Hierarchical: 부분에서 전체의 관계와 계층을 표현
  • Relational: 상관 관계와 연관을 표현
  • Temporal: 트렌트나 주기를 표현
  • Spatial: 오버레이나 왜곡을 사용하여 공간상의 패턴을 표현

Bar Chart

카테고리 사이의 정량적 차이를 표현합니다.

  • Marks: Line (Bar)
  • Attribute: Size (Length or Height)
  • Annotation: tick mark, grid line 활용하면 각 bar의 값을 직접 표현아지 않아도 됨
  • Variantions & Alternatives: 시간에 따른 변화를 보여 줄 수 있음

Clustered Bar Chart

Secondary 카테고리 비교 가능

Bullet Chart

Bar chart에 추가로 배경 색상 등을 활용하여 추가 해석 요소를 제공

Waterfall Chart

총량이 어떻게 이루어져있는지를 보여줌. 각 컴포넌트의 상승값과 하강값의 차이와 함께 총량을 표시 일반적으로 서로다른 항목의 수입과 지출로 부터 이익을 계산하는데 사용

  • Annotation: 일반적으로 값을 직저표시

Radar Chart

Polar Chart

Connected Dot Plot

부분 비교를 통해 2차 카테고리 비교 가능. 선을 연결하여 그 길이를 통해 차이를 표현. 2차 카테고리의 구분은 색상을 사용.

Pictogram

정량적 카운트를 표시하기 위해 기호 또는 그림의 형태로 점 표시를 반복합니다.

Propotional Symbol Chart

비교하는 것은 쉽지 않으므로 다양한 크기를 가질 때 적합합니다.

비교: Word Cloud

비교: Heat Map

비교: Matrix Chart

두 범주형, 이산형 데이터의 정량적 차원의 교차점에 걸쳐 정량적 값을 표시합니다.

분포: Dot Plot

분포: Beeswarm Plot

양적 척도를 따라 배치된 점 표시(일반적으로 작은 원)로 구성합니다. 점은 유사한 값 위치를 점을 정량적 축 기준선 위와 아래의 공간을 사용하여 균일하게 배치합니다.

분포: Histogram

분포: Density Plot

분포: Box-And-Whisker Plot

부분/전체: Pie Chart

부분/전체: Waffle Chart

여러 구성 범주에 대한 수량 비율이 전체를 구성하는 방법을 보여 줍니다.

부분/전체: Stacked Bar Chart

부분/전체: Diverging Bar Chart

각 성분 범주 비율의 양적 비율에 따라 크기를 결정하고 별도의 막대로 표시합니다.

부분/전체: Marimekko Chart

높이와 너비 모두에 대한 크기 변동이 있는 2차원 쌓인 막대형 관리도 입니다.

계층: Treemap

각 범주의 양적 값의 계층적 표시를 제공합니다.

계층: Sunburst Chart

범주형 차원의 여러 계층에 걸쳐 계층적 관계와 부분 대 부분 관계를 표시합니다

계층: Dendrogram

범주형 차원의 여러 계층 간의 계층적 관계를 표시하는 노드-링크 다이어그램입니다.

계층: Venn diagram

여러 집합의 집합 및 관계를 보여줍니다.

관계: Scatter Plot

관계: Bubble Plot

서로 다른 범주 항목에 대한 세 개의 양적 변수 간의 관계를 표시합니다.

연결: Node–link diagrams

항목 간의 연결을 통해 관계를 표시합니다.

연결: Sankey diagram

범주형 구성 및 서로 다른 주요 순서 차원 간의 양적 관계 흐름을 표시합니다.

연결: Chord Diagram

범주 항목 간 및 범주 항목 내의 연결을 통해 관계를 표시합니다.

경향: Line Chart

경향: Bump chart

서로 다른 범주형 항목에 대해 시간이 지남에 따라 양적 값이 어떻게 변화했는지를 보여줍니다.

경향: Slope Graph

서로 다른 범주 항목에 대해 두 시점에 걸쳐 양적 값이 어떻게 변화했는지 보여줍니다.

경향: Connected Scatter Plot

시간 경과에 따른 두 양적 측도 간의 관계를 표시합니다.

경향: Area Chart

경향: Stacked Area Chart

경향: Stream Graph

여러 범주형 항목에 대해 시간이 지남에 따라 양적 값이 어떻게 변화했는지 보여줍니다.

시차: Gantt Chart

여러 범주형 항목에 대한 시간 기반 구간을 표시합니다.

활동: Instance Chart

서로 다른 범주형 항목에 대한 시간 기반 이벤트를 표시합니다.

Overlays: Choropleth Map

구별되고 정의 가능한 공간 영역에 대한 정량적 값을 표시합니다.

Overlays: Isarithmic(등축성) Map

동일한 정량적 분류를 공유하는 맵에 고유한 공간 표면을 표시합니다.

Overlays: Proportional Symbol Map

지도의 위치에 대한 양적 값이 표시됩니다.

Overlays: Prism Map

맵의 위치에 대한 양적 값을 표시합니다. 값은 3D 막대의 비례적 크기의 선을 통해 표현합니다.

Overlays: Dot Map

점 표시를 사용하여 특정 지리적 좌표에 데이터 항목을 표시합니다.

Overlays: Flow Map

공간 영역에 걸친 현상들 사이의 움직임 또는 연결의 특성을 보여줍니다.

변형: Area Cartogram

정의 가능한 별개의 공간 영역과 관련된 양적 값을 지도에 표시합니다.

변형: Dorling Cartogram

지도에서 정의 가능한 별개의 공간 영역과 관련된 양적 값을 표시합니다.

변형: Grid Map

지도에서 구별되고 정의 가능한 공간 영역과 관련된 양적 값을 표시합니다. 각 지리적 영역(또는 통계적으로 일관된 공간 간격, '빈'이라고 함)은 타일이라고도 하는 고정 크기의 균일한 모양으로 표시됩니다.
